# Sistema Web Iron Max

Sistema web de gestão empresarial desenvolvido para a empresa *** , como parte de uma atividade universitária do curso de Análise e Desenvolvimento de Sistemas da Universidade Paulista — UNIP.

## Funcionalidades

- Cadastro, listagem, edição e exclusão de clientes
- Cadastro, listagem, edição e exclusão de funcionários
- Cadastro, listagem, edição e exclusão de serviços
- Dashboard com indicadores gerais
- Busca nas tabelas
- Armazenamento local dos dados com localStorage

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- Bootstrap Icons
- localStorage

## Observação

Este projeto é um protótipo funcional. Os dados são salvos localmente no navegador, sem utilização de banco de dados externo nesta versão.
